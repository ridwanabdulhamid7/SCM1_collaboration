SCM1A Group Collaboration Project

A demonstration of teamwork using Git and GitHub

About This Project

This website was built as part of our Software Configuration Management (SCM1A) group assignment. The purpose of this project is to practice real-world collaboration using Git and GitHub — including branching, committing, and submitting pull requests as a team.

Our Purpose

Each member of the group independently built their own profile page using a separate Git branch and pull request, then merged their work into this shared repository. This structure allowed all five of us to work at the same time without conflicts, while still contributing to one unified project.
